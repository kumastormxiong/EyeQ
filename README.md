# EyeQ 视觉AI助手

## 项目简介
EyeQ 是一款基于多模态大模型的智能视觉助手，支持屏幕区域截图+文本提问，结合图像与自然语言理解，助你高效获取屏幕内容的智能解读与问答。

## 主要功能
- **区域截图提问**：快捷键截取任意屏幕区域，自动上传并结合文本提问。
- **多模态问答**：支持图片+文字输入，AI智能理解并作答。
- **对话历史保存**：所有提问与AI回复自动本地保存，便于回顾。
- **美观深色UI**：全界面深色风格，输入栏、气泡、弹窗风格统一。
- **快捷键操作**：支持自定义截图、窗口呼出等快捷键。
- **信息弹窗**：H键一键查看使用指引与作者信息。

## 安装与运行
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   # 或手动安装 PyQt5 keyboard pyautogui requests pyperclip pillow markdown2
   ```
2. 配置API密钥：
   - 编辑 `config.ini`，在 [API] 部分填写你的大模型API密钥。
3. 启动程序：
   ```bash
   python main.py
   ```

## 界面亮点
- 聊天气泡：左问右答，支持Markdown渲染。
- 输入栏：深色背景，白色文字，风格统一。
- 聊天历史区：无滚动条干扰，极简美观。
- 信息弹窗：快捷键H，展示指引与作者。
